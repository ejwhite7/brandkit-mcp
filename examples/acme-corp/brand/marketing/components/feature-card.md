# Feature Card

Card component for displaying product features on the marketing website.

## Usage

Use in a grid layout (3 or 4 columns) to showcase product features.
Each card should highlight a single benefit or feature.

## Specifications

- Border radius: 12px
- Shadow: 0 4px 24px rgba(0, 0, 0, 0.08)
- Padding: 2rem
- Icon size: 48px
- Title: --font-family-heading at --font-size-xl, --font-weight-semibold
- Description: --font-family-primary at --font-size-base, --color-neutral-600
