# Sidebar Navigation

Primary navigation component for the product application.

## Usage

Always present in the product app layout. Shows main navigation items,
user account, and workspace switcher.

## Specifications

- Width: 260px (expanded), 64px (collapsed)
- Background: --color-sidebar-bg
- Text: --color-sidebar-text
- Active item: --color-accent with left border indicator
- Icon size: 20px
- Item padding: 8px 16px
- Transition: width 200ms ease-in-out
