# Data Table

Table component for displaying structured data in the product application.

## Usage

Use for any tabular data display. Supports sorting, filtering, and pagination.

## Specifications

- Header: --font-weight-semibold, --font-size-sm, --color-neutral-600
- Cell padding: 12px 16px (default), 8px 12px (compact)
- Row hover: --color-surface-raised
- Border: 1px solid --color-border
- Sortable columns show directional arrow icons
